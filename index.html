Projects

Brick Breaker

This is the classic game of Brick Breaker, in which the player's objective is to strike every brick with the ball. The catch is that the ball must not pass the paddle moving downward.

This game was created using Java for both the logic and the GUI.

Snake

This is another classic most of you probably know. If you are not familiar I will give you a quick explanation of this very simple game.

You, the snake, must eat each of the bugs that are placed on the board. As you eat more bugs, the snake gets longer. Avoid running into yourself or the game will end!

Drum Set

This webpage is a keyboard drum set using just a few of the keys on your board. As you play the keys, you will see them transform slightly along with the accompanying sound play.

This webpage was created using html, css and javaScript.
